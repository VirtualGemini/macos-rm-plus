# macos-rm-plus

`macos-rm-plus` is a macOS command-line tool. The core command is `rmp`.
It will move files and directories to the system Trash instead of permanently deleting them.

The project is currently in the scaffolding phase. The `rmp` executable intentionally exits with
an error until the first implementation ticket is complete.
